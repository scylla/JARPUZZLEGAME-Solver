# JARPUZZLEGAME-Solver
a java based solution generator for jar puzzle where the goal is to get the required configuration for jars of given capacity

input format is :

n -- number of jars
n space separated values for capacity of each jar
n space separated values for initial state of each jar
x the desired value to be obtained in a jar 

compile as
javac FillTheJarPuzzle.java

run as 
java FillTheJarPuzzle < input.txt
